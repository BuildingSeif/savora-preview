# Savora Mockup

THE official Savora/OptiCook app design mockup. Updated March 27 with Jerome + Nicolas feedback.

## Important
- `index.html` is the ONLY authoritative design reference for the OptiCook mobile app
- DO NOT modify without explicit approval — this is the approved design
- GitHub: BuildingSeif/savora-preview

## Design Direction
- Warm, welcoming glassmorphic, Apple-native feel (warmer than original dark heritage book)
- Terracotta accent color
- Frosted glass cards, buttery smooth animations
- Person cooking visible on onboarding (not hidden by button)
- Recipe count: "plus de 1 400 000 recettes" (not exact number)
- Recipe origins indicated via country flags
- Cook mode: magnifying glass icon for text resize (not enlarged by default)
- Language selector on first onboarding page
- Auto-carousel on onboarding panels
- See ~/Projects/Optirecipe for the backend this app will connect to
